# ziwei-astrology-tool
研究紫微斗數用萬用命盤產生
這是一套專為紫微斗數研究者設計的**純 HTML 離線版命盤產生器**。本程式無需安裝任何軟體或 Python 環境，只要透過任何電腦或平板的瀏覽器（如 Chrome、Edge、Safari）點開即可使用。

---

## 🌟 核心功能亮點

1. **純前端離線運行**：單一 `.html` 檔案，無需網路與資料庫，保障資料隱私。
2. **四段式圖層切換**：可隨時切換顯示層級，從最簡潔的 14 主星基本盤，到包含生年四化、十二宮位、六吉六煞與祿馬的進階盤。
3. **13 碼狀態還原系統**：獨創的固定長度 13 碼生成系統，只需一串數字即可完美保存並還原所有排盤設定。
4. **直式 / 橫式一鍵切換**：內建「直式 (A4)」與「橫式 (PPT/螢幕分享)」兩種版面，排版討論更方便。
5. **多行備註與完美列印**：支援多行備註，打字碰到邊界自動斷行不跑版；列印時自動隱藏操作面板，並鎖定精準高度，確保滿版輸出。

## 💻 介面與操作步驟

### 步驟一：選擇顯示圖層與版面方向
在畫面最上方，可依據您的研究需求進行設定：
*   **圖層切換**：
    *   **Layer 1 (基本盤)**：僅顯示 14 主星配置。（適合剛入門排盤練習）
    *   **Layer 2 (+四化)**：疊加顯示生年四化的 `[祿]、[權]、[科]、[忌]` 彩色標籤。
    *   **Layer 3 (+十二宮)**：疊加上命宮、兄弟、夫妻等 12 宮位名稱。
    *   **Layer 4 (+吉煞祿馬)**：排入六吉星、六煞星、祿存星與天馬星，展現完整盤象。
*   **版面方向**：
    *   **直式 (A4)**：適合列印成紙本檔案閱讀。
    *   **橫式 (PPT)**：適合在螢幕上觀看、螢幕截圖貼入 Word 或 PowerPoint 簡報中討論。

### 步驟二：輸入排盤參數
只要您更改下拉選單，下方的命盤就會**全自動即時更新**：
- **表號** (1~12)：決定 14 主星的基礎分佈（紫微星所在的宮位）。
- **年干** (甲~癸)：影響生年四化、祿存、擎羊、陀羅、天魁、天鉞的排定。
- **命宮** (子~亥)：決定十二宮位的起始位置。
- **年支** (子~亥)：影響火星、鈴星、天馬的排定。
- **生月** (1~12)：影響左輔、右弼的排定。
- **生時** (子~亥)：影響文昌、文曲、地空、地劫以及火星、鈴星的排定。

### 步驟三：自訂文字與列印
- **姓名與備註**：您可以在上方欄位輸入命盤主人的姓名。**備註欄支援「多行輸入（按 Enter 換行）」**，文字若太長會自動根據中宮邊界換行，絕對不會破壞版面。
- **列印輸出**：排盤完成後，點擊右下角的 **「🖨️ 列印命盤 (Ctrl+P)」**，系統會自動根據您的「版面方向」將印表機切換為直式或橫式，請確認紙張大小為 `A4`、邊界為 `預設`，即可印出漂亮的研究用紙。

## 🔢 生成碼 (13 碼) 系統詳解

為了讓研究者能夠快速記錄與分享命盤設定，本系統內建了**「生成碼還原」**功能。

### 生成碼格式
命盤中宮右上角的 13 位數字，是由您的設定自動拼湊而成的。格式為：
> `T(2碼) + S(2碼) + D(2碼) + Y(2碼) + M(2碼) + H(2碼) + L(1碼)`

*   **T (表號)**：`01` ~ `12`
*   **S (年干)**：`01`(甲) ~ `10`(癸)
*   **D (命宮)**：`01`(子) ~ `12`(亥)
*   **Y (年支)**：`01`(子) ~ `12`(亥)
*   **M (生月)**：`01` ~ `12`
*   **H (生時)**：`01`(子) ~ `12`(亥)
*   **L (圖層)**：`1` ~ `4`

*(註：若您選擇較低層級的圖層，系統會將不需要的參數自動補零為 `00` 以保持格式統一)*

### 如何還原命盤？
1. 複製命盤上的 13 碼數字（例如：`0307121005124`）。
2. 在網頁黃色的**「還原碼」**輸入框中貼上這串數字。
3. 點擊 **「🔄 還原」**，所有的下拉選單、版面與圖層設定就會瞬間歸位！

## ⚠️ 注意事項與建議

1. **天馬星流派**：本程式內建的天馬星排法為**「年馬」**（依出生年支排定）。
2. **存檔建議**：若想在電腦中永久保存特定命盤，建議在點擊列印後，將印表機目的地改為**「另存為 PDF」**，即可獲得格式完美的數位命盤檔案。
